# Text Extraction From Image

This project is a simple and efficient **Flask-based web application** that extracts text from images using an OCR model.
Users can upload an image, and the application processes it to display the extracted text.

---

## Features

* Upload images (PNG, JPG, JPEG)
* Extract text using a trained OCR model
* Simple and easy-to-use web interface
* Built using Flask (Python)
* Sample images included for testing

---

## Technologies Used

* **Python**
* **Flask**
* **OCR Model / Deep Learning**
* **OpenCV**
* **HTML / CSS**

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### 2. Install required packages

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python app.py
```

Open your browser and go to:

```
http://127.0.0.1:5000/
```

---

## How to Use

1. Run the app
2. Upload an image containing text
3. The extracted text will be displayed on the screen
4. Copy or save the text as needed
